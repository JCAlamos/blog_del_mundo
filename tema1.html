Programando por línea de comandos

La programación por línea de comandos es una primera forma de lograr construir códigos muy simples o avanzados usando técnicas de escritura que al día de hoy siguen siendo efectivas cuando desarrollamos código. Siempre es mejor aprender los fundamentos para de ahí poder crecer y lograr más conocimiento, el fundamento básico para realizar cualquier programa es aprender a programar mediante el método de programación elemental el cual es utilizando la escritura por líneas de comandos independientemente del lenguaje de programación que se esté utilizando para realizar un código, después conoceremos un poco acerca de estos lenguajes y cuáles son sus ventajas y desventajas, por el momento solamente es importante tener en consideración que si aprendes algo llamado lógica de programación, no importará el lenguaje utilizado porque siempre serás capaz de traducirlo a código que realice las acciones que tú desees.
